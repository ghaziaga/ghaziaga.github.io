Penggunaan

Jenis API yang akan digunakan adalah Rest API. REST API sendiri merupakan salah satu desain arsitektur yang terdapat didalam API itu sendiri. Dan cara kerja RESTful API, yaitu klien REST akan mengakses data / sumber daya di server REST tempat setiap sumber daya berada. Jika tidak, data/resource akan dibedakan berdasarkan ID global atau URI (Universal Resource Identifier). Oleh karena itu, data yang disediakan oleh server REST di masa mendatang bisa dalam format teks, JSON atau XML. Saat ini, format yang paling populer dan banyak digunakan adalah format JSON.

Langkah Penggunaan

  • Method API yang bisa digunakan adalah GET untuk mendapatkan data-data pegawai diseluruh divisi
  • hrdemployee.herokuapp.com merupakan tempat SDM-API berada, disana tersimpan semua data Pegawai dengan format Json
  • Ketikkan https://hrdemployee.herokuapp.com/employee pada browser untuk melihat seluruh data pegawai
  • Ketikkan https://hrdemployee.herokuapp.com/employee/name/{name} pada browser untuk mengambil data per pegawai dengan sumber nama
  • Klik Disini untuk melihat Deployer lebih lengkap
